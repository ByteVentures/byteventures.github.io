Internet Ads: Now And Then

It should come at no surprise to everyone that advertisements on the internet are getting pretty ridiculous these days. They are getting out of hand. You don’t just see ads while browsing the internet, through web pages but also in almost every single app that connects to the internet. YouTube, Instagram, Facebook, Reddit to name a few.

Now we get it, Advertisements are the main sources of income for these companies, but that truly doesn’t justify the amount of advertisements that are thrown at users these days. Not to mention the types of Ads targeted.

It didn’t use to be like this of course. Remember the good ol’ internet days from 2012 to 15-16 ? Where the ads didn’t use to be so daunting ? And the content on platforms like YouTube used to actually be good? (But I don’t want to get into that now).

I have one main goal here : That is, by the end of this blog post, I want you to encounter the least amount of ads (and more than just ads) on the internet, to not only save time but also your peace of mind.

DNS Servers

Pihole DNS

Let’s get the more complicated stuff out of the way first. If you’ve never heard of Pihole before, it is basically a self-hosted DNS server that can block ads and websites. Its fully customizable and is really fast as it’s hosted locally in your network.

Setting up Pihole isn’t straightforward if you are new to servers and linux, but if you are interested or are already experienced with linux servers then check out my Pihole Post in my Projects Section.

DNS Servers

To maximize adblocking, it is best to do it at the DNS level. This is of course done by setting up DNS servers.

  • There are a few ways to implement this:

If you have a Router, you could log in to it, find the DNS settings (It differs form router to router; look up yours if you are unsure) and change the DNS servers.

If you don’t have a router and are just using devices through Mobile Data, or maybe you don’t have access to your router, then you can make changes to the DNS individually.

If you have an Android Device running Android 9 +, you could use the Private DNS setting which can be found under **Network Settings.

You could make changes to your Network Adapter’s DNS Settings on your PC or MAC.

Keep in mind the Private DNS feature uses DNS addresses a bit differently than what you might be accustomed to seeing.

Recommended DNS Servers

Adguard DNS:

  • 94.140.14.14 and 94.140.15.15

Adguard has a plethora of servers meant for different purposes. Hence, I recommend checking out this website to see what best suits your needs.

Private DNS:

  • dns.adguard.com

Cisco’s OpenDNS is also a good option if you want content filtering.

There are also Ad blocking apps that you can install on your phone. If your phone is rooted, you can take things to another level, but that is out of the scope of this post.

Brave Browser

Brave Browser is hands down the best browser to use if you want ad blocking throughout the internet.

It not only blocks advertisements, pop-up ads, malicious web pages, but also blocks ads on websites like YouTube.

Now admittedly, it does have a few things that are unnecessary for most people like the crypto wallet etc. However, you can disable those in the settings. Oh, and while you’re messing around in the settings, make sure to change the default search engine to something you like (Google). With a little bit of tweaking Brave Browser can be everything you need.

As browsers like Google Chrome on android don’t support ad blocking or extensions, Brave for Android or IOS is a really good option.

Extensions

Let’s say you don’t wanna use the Brave Browser for whatever reason and want to use something like Chrome or Firefox. Then extensions are your best bet. Here’s to name a few:

  • uBlock Origin
  • AdBlock
  • AdBlock Plus

There are many more out there and again it’s hard to list every single one so head out there and find one that best suits your needs.

Modified Apps

By using the methods mentioned above you can only block ads on websites and maby inside a few apps if you used adguard or pihole but it’s very rare.

Hence, blocking ads on apps individually by installing a modified version of them is the more reliable option.

  • Now I know we’re walking on real thin ice here. People aren’t really comfortable installing Unknown Apps, however if you install them from trusted projects run by really good and well known communities then it’s not an issue.

Instander

A modified version of Instagram with more than just ad blocking. It also has features for Anonimity and options to view pictures in higher quality which is absolutely amazing for photographers.

IMO instragram has been ruined by reels. It’s not what it used to be, but then again what is ? Although you can’t really remove reels from it, Instander does give you an option to stop autoplaying videos which will atleast help retain your peace of mind.

  • The projected was started by thedise and you can download it from here

YouTube Vanced and YouTube Revanced

The ads on youtube need no introduction. Unbelievably ridiculous, unrelated, unskippable and unfathomable. I can go on, you get the point. YouTube Vanced was created to not only remove ads but also to implement user requested features which YouTube doesn’t care about.

  • Features like the
    • Old Video Quality Layout
    • Brightness and Volume Slider
    • The Return YouTube Dislike API
    • Sponsor Block API

Unfortunately the Vanced project was discontinued due to a DMCA by google. However, a similar project called Revanced was started to continue the legacy of Vanced.

Keep in mind the Revanced Project at the time of writing this post is still relatively new and thus will contain bugs. You can get the manager from here

  • Revanced has patches also for YouTube Music and Reddit.

Although the Vanced Project has been discontinued you can still find the manager on the net and install it in case Vanced works better than Revanced. Test and try it out for yourself.

Pro Tip: It is better to install the patched apks using the Non-Root method alongside Micro-G even if you have a rooted device.

Xmanager for Spotify

  • A trusted source for patched spotify apks.
  • It has features of the premium version of spotify and features like an all black theme for amoled devices.

Conclusion

All the things mentioned in this post is what I use in my day-to-day life.

Although there are a lot more projects out there that do various tasks, these fulfill the ad blocking needs for most people. The internet is worth exploring for projects like these. There are plenty out there that might just have the fix for you !
