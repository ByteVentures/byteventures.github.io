Unlocking Huawei B310s-927 + OVPN

Mahir | Aug 14, 2022 min read

What Does Unlocking Mean ?

Here, unlocking means to get access to features that were locked (restricted) by the provider.In this case the provider is the ISP and not the manufacturer (Huawei).

Why Is Unlocking Necessary ?

The Huawei B310s-927 is a very famous router provided by ISPs for mobile networks. Meaning, you could use a SIM card to get data and have access to features found in traditional routers. However the ISP modifies the router to only accept their SIM cards and not others’. This isn’t a big deal if you’re in their area of coverage.

If you were to move out of their area of coverage then the router is deemed unusable (in relative terms). Unlocking the router however allows you to put any SIM card form anywhere around the world and use it without any restrictions.

Some Backstory

Unlocking this router was one of my first proper IT projects.I first started working on this project about 4 years ago. The procedures I had used for unlocking the router back then were a bit complicated. I had to solder a Data/USB cable to the motherboard of the router to connect it to my computer which allowed me to transfer necessary files for downgrading the firmware. At that time I had little to no idea what I was doing as I was pretty new to it and was mostly scouring the internet for tutorials and articles.

Nowadays the entire process is pretty starightforward and can be done within 20 minutes by using protocols like telnet to connect to the router.

I won’t get into the details on how to unlock the router as there are countless tutorials on youtube and articles on the internet that already cover it.

AIO Tools

This is something that I came across recently. A few developers created an AIO (All In One) Tool that cointains features like OpenVPN (OVPN) sTunnel, SoftEther and ShadowSocksR. The tool can be installed within a few minutes using a script and installs a seperate web page that can be accessed through a specified port.

This is an absolute game changer as by default the router doesn’t come with any VPN functionalities let alone OVPN and other Tunneling features.

Installation Procedures

During my research I wasn’t able to find tutorials that worked right out of the box as most if not all tutorials weren’t in english. However after a lot of trial and error and seeing what works best this is what worked for me:

Installing A Modified Firmware

Assuming that you already have your PC and Router connected via Ethernet and can access the router, the first step is to install a modified firmware (.bin file) which you can get from here and install it through a Local Update. I recommend you only install the .bin file from here and nothing else.

All the credit for files go to their respective owners/ developers.

Installing The AIO Tool

Installing the modified firmware is necessary as in my experience the AIO install doesn’t work without it. Once you have installed the modified firmware you can proceed to install the AIO tool for which the link can be found here.

Make sure you’re connected to the router (do not change the default IP Address) and run the script.

If everything works correctly it should say connected and proceed with the install.

Accessing The AIO Tool

The web interface can be accessed by visiting 192.168.8.1:8989 ; the default port is 8989.It is also necessary to have internet access in order to open the AIO Tool webpage but that’s not the case for the router’s web page (which is obvious). You can add your config files, upload them and connect to your desired protocol.

Keep in mind that you can only upload 1 config at a time. Incase you need to upload a different config, you have to remove the previous config from the Configuration Remover.

Conclusion

By installing the modified firmware one could gain access to hidden settings that allows the user to select LTE / Network Bands which is handy to have in different corners of the world. AIO tools allows you to connect to your desired VPN Provider and much more.

By having these options unlocked / installed it makes the B310s-927 a perfect router to take with you on journies away from home. It not only gets you better reception (Antennae play a huge role) but also keeps you safe online by having a VPN connection if you want to connect to public networks !

The only thing this router lacks which could make a huge difference in its usability is a wirless repeater mode. If it did, it could mean that the router could connect to other networks (Wi-Fi) without the need of an Ethernet cable. Connecting to public networks with an ethernet cable isn’t really a viable option.
