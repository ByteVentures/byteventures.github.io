Padavan Firmware for Mi 3C Router

Mahir | Jan 16, 2023 min read

What is Padavan ?

Padavan firmware is a custom firmware for wireless routers. It is based on the open-source firmware of Asuswrt, which is the firmware developed by Asus for their routers. Padavan firmware is compatible with a wide range of routers and provides many advanced features, such as support for various VPN protocols, advanced QoS controls, and support for custom scripts and plugins.

It also aims to improve the performance and security of the routers. It is a third-party firmware and is not officially supported by the router’s manufacturer.

  • The firmware is developed by a Chinese developer and is offered as a free and open-source software.

Custom Firmwares for Mi 3C Router:

The Xiaomi Mi 3C router is a popular router that is known for its low cost and high performance. There are several custom firmware options available for the Mi 3C router. Some popular options include:

Padavan firmware

As I mentioned earlier, Padavan firmware is a custom firmware that is based on the open-source firmware of Asuswrt. It offers many advanced features and is compatible with a wide range of routers, including the Mi 3C.

OpenWRT firmware

OpenWRT is an open-source firmware that is designed for embedded devices such as routers. It offers a wide range of features and is highly customizable. There are several versions of OpenWRT firmware available for the Mi 3C router.

DD-WRT firmware

DD-WRT is another open-source firmware that is designed for wireless routers. It offers a wide range of features and is highly customizable. There are several versions of DD-WRT firmware available for the Mi 3C router.

X-WRT firmware

This is an open-source firmware that is based on OpenWRT, it provides more features and options than the original firmware, but it’s not as popular as the others.

Why Padavan and not others ?

There is nothing stopping you from installing other custom firmwares; however, from my testing, OpenWRT has a couple of bugs, and some features that you’d expect to work right out of the box don’t. Thus far, Padavan is the only firmware that has all the features that you’d expect, and they work seamlessly.

Installation Guide:

Here’s the reference guide on Github.

The guide will take you through, step by step on how to get any custom firmware up and running on your router initially. Later on in, the guide will be more specific to the Padavan Firmware.

It is worth mentioning that this guide can also be used for other routers as well. However, only as a reference.

Prerequisites

Have a wired connection to the router from your PC. Assign the Ethernet port a static IP such as 192.168.31.2.

Please Remember that you may need to change the static IP frequently as you proceed with the installation.

You may also need internet access for reference. (If you have a Wi-Fi card, it may be helpful)

Some basic Networking knowledge. Although the guide will give you all the information you need, it helps to know what you are doing.

A Linux machine. Don’t worry I’ll show you a method that doesn’t require VMs.

Setting Up linux

You don’t need a VM to run Linux. You can install it using WSL. If you haven’t heard about it, WSL stands for Windows Subsystem for Linux. It is a feature in Windows 10 or higher that allows users to run Linux command-line tools and utilities natively on Windows. This means that users can use Linux commands and utilities, such as Bash, directly in the Windows Command Prompt or PowerShell, without the need for a virtual machine or dual-booting.

Do note that you need Windows 10 2004 or higher for this to work

You can check the official documentation for a better understanding.

You also need to enable virtualization in your BIOS.

Once you have enabled virtualization, open a terminal or command prompt window with Administrator Privileges and type the following command:

wsl --install

Reboot your PC as the prompt says and after reboot it’ll continue installation. Proceed with the usual installation stuff, setting up username and password and so on.

Once Ubuntu is up and running we can proceed further.

Gaining Telnet Access

For this step, I recommend following this video

You may need to install Telnet tools on your Linux instance, just bear that in mind.

Installing Breed Bootloader

Once you have Telnet access, you need to transfer the bootloader file into the /tmp directory of the router.

You can do this via Filezilla or any application of your choice.

  • Username: root
  • Password: root
  • Port: 21

Once the file is transferred, open a terminal or cmd. Login via telnet using:

telent 192.168.31.1

Leave the password blank and hit enter.

Once you have logged in, type in the following command to install BREED:

mtd -e Bootloader -r write /tmp/breed-mt7688-reset38.bin Bootloader

Wait for the installation to take place. Do not unplug your router. Wait for a couple of minutes to make sure the nstallation completes.

You can enter via breed by heading over to

192.168.1.1

Do note that if you have assigned your PC a static IP, then you need to change it accordingly. Set your Pc’s IP address to

192.168.1.100

Installing Padavan Firmware

From here on out the guide becomes specfic to Padvan Fimrware.

You can compile your own Padavan Firmware if you like, but I won’t be diving into that as we already have a pre-compiled firmware to work with. Firmware: MI-3C_3.4.3.9-099.trx.tar.gz (I did not create these files. The credit goes to the respective owners/creators.)

You do need to extract the firmware. You can extract it using your linux instance. I’ll be using Ubuntu that we installed earlier using wsl.

Paste the file into a working directory of Ubuntu. If you are using Windows 11, Linux should appear in the file explorer, and you can navigate to the directory.

If you are using Windows 10, then enter

\\wsl$\

In the file explorer address bar.

You can the extract the file by using the following command.

tar -xf MI-3C_3.4.3.9-099.trx.tar.gz

After extracting, upload the .tar firmware file to Breed as stated before and proceed with the installation.

If you can access the login page at -

192.168.2.1

(assuming that you have either set a static IP in the subnet or set to automatic IP) then you have successfully installed padavan.

The default username and password is admin and admin respectively. You should be able to log in into your router.

Changing language in Padavan Firmware.

By default, everything is in chinese.

You can change the language by heading over to the administration menu (5th menu from the wrench icon), System (First menu) and then scrolling all the way down to the last menu and selecting English from the drop-down.

Conclusion

If you have made it so far then congratulations. You now have a powerful custom firmware to meet all your networking needs on a simple home router !
